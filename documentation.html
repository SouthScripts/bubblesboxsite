Getting Started

Welcome to Bubbles Box, the ultimate Minecraft Box PvP server! This guide will help you get started with our server and make the most of your experience.

Connecting to the Server

To join our server, follow these steps:

  1. Launch Minecraft Java Edition (version 1.19.2 or later)
  2. Click on "Multiplayer" from the main menu
  3. Click "Add Server"
  4. Enter "Bubbles Box" as the server name
  5. Enter "play.bubblesbox.net" as the server address
  6. Click "Done" and then double-click on the server to join

Your First Login

When you first join the server, you'll spawn in our lobby area. Here you can:

  • Read the server rules
  • Choose a game mode
  • Select your kit
  • Join a match

Use the command /help to see a list of available commands.

Server Rules

To ensure a fair and enjoyable experience for all players, please follow these rules:

General Rules

  1. Be respectful to all players and staff
  2. No harassment, discrimination, or hate speech
  3. No spamming or excessive caps in chat
  4. No advertising other servers or services
  5. English only in main chat

Gameplay Rules

  1. No hacking, cheating, or using unfair advantages
  2. No exploiting bugs or glitches
  3. No teaming in solo game modes
  4. No combat logging (disconnecting during fights)
  5. No using macros or auto-clickers

Breaking these rules may result in warnings, temporary bans, or permanent bans depending on the severity and frequency of the offense.

Game Modes

Bubbles Box offers several game modes to suit different playstyles:

Classic Box PvP

The traditional Box PvP experience with standard kits and maps. Fight in a confined space and be the last player standing.

Command: /play classic

Team Battles

Form teams and compete in 2v2, 3v3, and 4v4 Box PvP matches. Coordinate with your teammates to defeat the opposing team.

Command: /play teams

Ranked Mode

Compete in ranked matches to climb the leaderboard. Win matches to gain ELO points and unlock exclusive rewards.

Command: /play ranked

Practice Mode

Hone your skills in a non-competitive environment. Practice your combos, movement, and strategies without affecting your stats.

Command: /play practice

Commands

Here are some useful commands you can use on our server:

General Commands

  • /help - Shows a list of available commands
  • /rules - Displays the server rules
  • /spawn - Teleports you to the spawn area
  • /lobby - Returns you to the main lobby

Game Commands

  • /play [mode] - Joins a specific game mode
  • /kit [name] - Selects a specific kit
  • /stats - Shows your player statistics
  • /leaderboard - Displays the server leaderboard

Social Commands

  • /msg [player] [message] - Sends a private message to a player
  • /r [message] - Replies to the last private message
  • /friend add [player] - Sends a friend request
  • /party invite [player] - Invites a player to your party

FAQ

Frequently Asked Questions

What versions of Minecraft are supported?

Our server supports Minecraft Java Edition versions 1.19.2 to 1.20.4.

Can I play on Bedrock Edition?

Currently, our server only supports Java Edition. We may add Bedrock support in the future.

How do I report a player?

Use the command /report [player] [reason] in-game or join our Discord server and open a ticket.

How do I apply for staff?

Staff applications are opened periodically. Check our Discord announcements for information about when applications are open.

Is there a resource pack?

Yes, we have an optional resource pack that enhances the visual experience. You'll be prompted to download it when joining the server.

How do I create a team?

Use the command /team create [name] to create a team, then invite players with /team invite [player].
